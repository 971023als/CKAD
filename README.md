## 📝 학습 팁
- CKAD 시험은 CLI 속도가 중요하므로, 아래와 같은 팁을 연습하세요:
  - `kubectl` 명령어 자동완성 설정

- [Mock Exam 연습](https://killer.sh/)을 통해 시험 환경에 익숙해지세요.
